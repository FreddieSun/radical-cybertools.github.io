Ensemble MD Toolkit

Under Development

The EnsembleMD Toolkit (EnMDTK) project is a loose collection of RADICAL-Pilot based tools, programming libraries and APIs that simplify and abstract common tasks, patters and tools for molecular dynamics workflows. All of these tools encapsulate at least one of the commonly used MD engines, GROMACS, AMBER and NAMD.

Join the mailinglist at https://groups.google.com/d/forum/ensemblemd-toolkit

Ensemble API

Provides a command line interface to run multiple Molecular Dynamics simulations using Radical pilot as the underlying API coupled with Analysis tools.

Multi-Task Multi-Stage (MTMS)

The MTMS library allows a user to execute a multitude of tasks (parallel pipelines) with a certain number of stages (steps). This can be done by simple specifing an application to run for every instance, a number of input files (which become tasks) and the number of stages to perform.

Github Project

Binding Affinity Calculation (HT-BAC)

HT-BAC or "High-Throughput Binding Affinity Calculator' is a set of NAMD-based tools to automate the workflow involved in molecular dynamics based binding affinity calculations. It was originally designed for the simulation of HIV-1 protease (PR) bound to a variety of ligands but has been extended for use with a wide range of pathologies such as the Abl tyrosine kinase and human epidermal growth factor receptor (EGFR).
